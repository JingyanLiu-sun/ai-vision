## 概述

N 皇后问题要求在 N×N 棋盘上放置 N 个皇后，使得任意两个皇后都不互相攻击。皇后的攻击范围包括行、列和两条对角线。

## 算法

使用带约束检查的回溯：

- 维护集合：`列 columns`、`主对角 diag1 = r - c`、`副对角 diag2 = r + c`
- 对每一行 r 依次尝试列 c
- 若 `(c 不在 columns) && (r-c 不在 diag1) && (r+c 不在 diag2)` 则可放置；否则继续尝试下一列
- 若某行无法放置，回溯：移除上一行的皇后，尝试其下一列

## 可视化控件

- 棋盘大小：4–12
- 自动播放 / 暂停 / 步进 / 重置
- 高亮冲突位置：显示当前部分布局下被攻击的格子
- 查找全部解：找到一个解后是否继续搜索
- 速度调节：控制自动播放速率
- 搜索轨迹：显示尝试 / 放置 / 回溯事件

## 复杂度

时间复杂度随 N 指数增长，但通过约束剪枝，在中等规模下仍可较好地运行。
